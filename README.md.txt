# Amazon E-Commerce Sales EDA

## Objective
To analyze Amazon e-commerce sales data and extract business insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Sales distribution
- Order status analysis
- Top products and categories
- Revenue analysis
- KPI metrics

## Key Insights
- Delivered orders contribute the highest revenue
- Cancelled and returned orders have minimal impact
- Few categories generate most sales
